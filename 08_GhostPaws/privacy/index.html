개인정보 이용약관 지원

개인정보처리방침

시행일 및 최종 업데이트: 2026년 7월 12일

pungpungstudi(이하 “개발자”)는 모바일 게임 냥벽수비대: 고스트 포우즈(Ghost Paws)(이하 “게임”)의 개인정보 처리 방식을 다음과 같이 안내합니다.

현재 공개 예정 빌드 요약: 게임은 계정을 요구하지 않으며, 개인정보나 기기 데이터를 개발자 서버 또는 제3자에게 수집·전송·판매·공유하지 않습니다. 게임 진행 정보와 플레이 기록은 이용자의 기기에만 저장됩니다.

1. 적용 범위

본 방침은 Android 및 iOS용 게임에 적용됩니다. 앱 스토어, 운영체제, GitHub 지원 페이지 등 외부 서비스가 자체적으로 처리하는 정보에는 각 서비스의 개인정보 보호정책이 적용됩니다.

2. 개발자가 수집하거나 전송하는 정보

현재 배포 대상 빌드에서 개발자는 다음 정보를 수집하거나 기기 밖으로 전송하지 않습니다.

  • 이름, 이메일 주소, 전화번호, 생년월일 또는 계정 정보
  • 연락처, 사진, 카메라·마이크 데이터 또는 파일
  • 정확한 위치 또는 대략적인 위치
  • 광고 식별자, 기기 식별자, IP 주소 또는 쿠키
  • 결제 카드 정보, 구매 영수증 또는 거래 식별자
  • 분석, 광고 성과, 충돌 보고 또는 원격 진단 데이터

현재 게임에는 광고, 분석, 충돌 보고, 계정 로그인, 클라우드 저장 또는 실제 인앱 결제용 제3자 SDK가 포함되어 있지 않습니다.

3. 기기에 저장되는 데이터

게임 기능과 진행 복원을 위해 다음 데이터가 앱의 로컬 저장공간에 저장될 수 있습니다.

  • 물고기 쿠키 등 가상 재화 잔액과 재화 획득·사용 결과
  • 고양이, 무기, 유물, 맵, 스티커북 및 메타 업그레이드의 해금·보유 상태
  • 최고 기록, 런 결과, 보스 진행, 보상 사용 횟수와 게임 진행 상태
  • 음량, 자동 공격, 접근성, UI 배율 및 조작 설정
  • 저장 무결성 확인과 복구를 위한 저장본, 임시 파일 및 백업 파일
  • 밸런스와 오류 점검을 위한 로컬 플레이 이벤트 기록(JSONL 형식)

이 데이터는 개발자에게 전송되지 않으며 같은 기기에서 게임을 실행하고 저장 상태를 복구하는 데 사용됩니다. 운영체제의 기기 백업 기능이 앱 데이터를 복사할 수 있으며, 이 경우 처리는 Apple 또는 Google 등 해당 플랫폼의 설정과 정책을 따릅니다.

4. 권한

현재 게임 기능은 위치, 연락처, 카메라, 마이크, 사진 보관함 또는 추적 권한을 사용하지 않으며 해당 권한을 요청하는 흐름도 실행하지 않습니다. Android 배포 설정에서는 인터넷 권한도 비활성화되어 있습니다.

iOS 빌드 파일에 운영체제 호환을 위한 권한 설명 키가 포함될 수 있으나, 게임이 관련 API를 호출하거나 권한 요청을 표시하지 않는 한 데이터에 접근하지 않습니다.

5. 광고와 선택형 보상

게임 코드에는 향후 선택형 보상 광고를 연결할 수 있는 인터페이스가 있으나, 현재 배포 대상 빌드에는 실제 광고 제공자 또는 광고 SDK가 연결되어 있지 않습니다. 따라서 광고가 표시되지 않고 광고 관련 데이터도 처리되지 않습니다.

향후 광고 SDK를 포함하는 버전을 배포하는 경우, 적용 전에 본 방침과 스토어의 개인정보·Data safety 신고를 수정하고 필요한 동의 절차를 구현합니다. 광고 SDK가 처리할 수 있는 IP 주소, 광고 식별자, 광고 상호작용 및 진단 정보는 실제 SDK와 설정이 확정된 뒤 구체적으로 공개합니다.

6. 인앱 결제

현재 배포 대상 빌드는 실제 인앱 결제를 제공하지 않습니다. 개발용 결제 모형은 릴리스 빌드에서 비활성화됩니다. 향후 유료 상품을 제공할 경우 결제는 Apple App Store 또는 Google Play의 공식 결제 시스템을 통해 처리하며, 적용 전에 관련 정보 처리와 복원 방식을 본 방침에 반영합니다.

7. 데이터 보유 및 삭제

로컬 데이터는 이용자가 앱 데이터 또는 게임을 삭제할 때까지 기기에 남을 수 있습니다.

  • Android: 시스템 설정에서 앱 저장공간을 삭제하거나 게임을 제거합니다.
  • iOS: 게임을 삭제합니다. 기기 또는 iCloud 백업에 남은 사본은 Apple의 백업 설정에서 관리합니다.

게임에는 사용자 계정이나 개발자 서버가 없으므로 개발자가 원격으로 조회하거나 삭제할 서버 데이터는 없습니다. 삭제 전에 필요한 진행 정보는 이용자가 직접 보관해야 하며, 삭제된 로컬 데이터는 복구되지 않을 수 있습니다.

8. 아동의 개인정보

게임은 이용자의 연령을 확인하거나 아동의 개인정보를 의도적으로 수집하지 않습니다. 출시 국가와 스토어에서 정한 연령 등급 및 보호자 통제 기능이 적용될 수 있습니다. 향후 아동 대상 서비스 또는 데이터 처리 기능을 추가한다면 출시 전에 관련 정책과 동의 절차를 다시 검토합니다.

9. 보안

현재 데이터는 기기에만 저장되어 네트워크 전송 과정은 없습니다. 개발자는 임시 파일과 백업 저장본을 사용해 저장 실패와 손상을 줄이도록 설계하지만, 기기 고장, 운영체제 변경, 저장공간 삭제 또는 비공식 수정으로 인한 데이터 손실을 완전히 방지할 수는 없습니다.

10. 외부 서비스

게임 설치, 업데이트, 결제 또는 지원 페이지 사용 과정에서 플랫폼 사업자가 독립적으로 정보를 처리할 수 있습니다.

  • Google 개인정보처리방침

11. 방침 변경

게임 기능, SDK 또는 법적 요구사항이 변경되면 본 방침을 수정할 수 있습니다. 데이터 처리에 중대한 변경이 있는 경우 새 빌드 제출 전에 공개 페이지와 스토어 신고를 갱신하고, 필요한 경우 앱 안에서 추가 안내 또는 동의를 제공합니다.
