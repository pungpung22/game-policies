개인정보처리방침

시행일: 2026년 7월 13일 · 최종 업데이트: 2026년 7월 16일

pungpungstudi(이하 “개발자”)는 모바일 게임 냥벽수비대: 고스트 포우즈 (com.pungpungstudi.ghostpaws, 이하 “게임”)의 개인정보 처리 방식을 다음과 같이 안내합니다.

요약: 게임 진행도는 기기에 저장됩니다. 상점의 선택형 보상 광고를 위해 Google Mobile Ads SDK와 Google User Messaging Platform(UMP)을 사용하며, Google 및 광고 파트너가 광고 관련 데이터를 처리할 수 있습니다. 앱 업데이트 확인에는 Firebase Remote Config, 비정상 종료 진단에는 Firebase Crashlytics를 사용합니다.

1. 서비스와 적용 범위

게임은 Android와 iOS에서 진행도를 로컬에 저장하고 상점에서 선택형 보상 광고를 제공합니다. 앱 업데이트 알림에는 Firebase Remote Config, 비정상 종료 진단에는 Firebase Crashlytics를 사용합니다. 계정 로그인, Google Analytics, 클라우드 저장, 멀티플레이, 채팅, 사용자 제작 콘텐츠는 사용하지 않습니다. 앱 스토어, 운영체제, Google 광고 및 GitHub 지원 페이지에는 각 사업자의 정책이 적용됩니다.

2. 광고 제공 과정에서 처리될 수 있는 데이터

이용자가 보상 광고를 불러오거나 시청하면 지역, 동의, 기기 설정 및 노출 광고에 따라 Google과 광고 파트너가 다음 정보를 자동으로 처리할 수 있습니다.

  • 일반적인 위치 추정에 사용될 수 있는 IP 주소
  • 광고 ID 또는 앱 범위 식별자를 포함한 기기·계정 식별자
  • 앱 실행, 탭, 광고 노출, 동영상 시청 등의 광고·제품 상호작용
  • 비정상 종료, 실행 시간, 멈춤, 에너지 사용량 등의 진단·성능 정보
  • 이용자에게 표시된 광고 등 광고 데이터

이 정보는 광고 제공, 광고 측정·분석, 보안 및 부정행위 방지에 사용될 수 있으며 암호화된 연결로 Google에 전송됩니다. 개발자는 이 광고 데이터를 받는 별도 서버를 운영하지 않으며 개인정보를 직접 판매하지 않습니다. 자세한 내용은 Google 개인정보처리방침을 확인해 주세요.

3. 업데이트 확인과 비정상 종료 진단

Firebase Remote Config는 업데이트 설정을 전달하기 위해 앱 범위 Firebase 설치 식별자와 기술적인 요청 정보를 처리할 수 있습니다. Crashlytics는 비정상 종료 보고서, 스택 트레이스, 앱·OS 버전, 기기 모델, 프로세스 상태, 발생 시각 및 개발자가 추가한 진단 키를 처리할 수 있습니다.

게임은 현재 앱 버전, 플랫폼, 엔진 범위 같은 기술 값만 진단 키로 추가하며 계정 정보, 로컬 저장 데이터, 광고 ID 또는 정밀 위치를 오류 보고서에 의도적으로 첨부하지 않습니다. 이 정보는 업데이트 안내, 안정성 문제 진단과 앱 신뢰성 개선에 사용되며 Google의 Firebase 약관과 개인정보 문서에 따라 보관됩니다. 이 Firebase 프로젝트의 Google Analytics는 사용하지 않습니다.

4. 개인정보 선택과 광고 방식

게임은 필요한 지역에서 실행할 때마다 UMP로 최신 동의 상태를 확인합니다. Google이 개인정보 선택 양식을 제공하면 설정 또는 상점의 광고 개인정보 설정에서 다시 열 수 있습니다. Android/iOS 설정에서도 광고 식별자를 제한할 수 있습니다.

현재 게임은 Apple 앱 추적 투명성(ATT) 권한을 요청하지 않으며 해당 권한 없이 IDFA를 의도적으로 사용하지 않습니다. 향후 IDFA 기반 추적을 선택하면 ATT 안내와 App Privacy를 먼저 갱신합니다.

광고 시청은 선택 사항입니다. 광고 1회 완료 시 생선쿠키 20개, UTC 날짜 기준 하루 최대 3회까지 지급합니다. 완료 전에 닫으면 재화를 지급하지 않습니다. 배너, 전면, 앱 시작, 강제 부활, 레벨업 리롤 광고는 사용하지 않습니다.

5. 기기에 저장되는 데이터

  • 가상 재화 잔액, 누적 획득량, 광고 완료 날짜·오늘/전체 횟수
  • 고양이, 무기, 유물, 맵, 스티커 및 메타 업그레이드 상태
  • 최고 기록, 런 결과, 보스 진행과 보상 사용 기록
  • 음량, 자동 공격, 접근성, UI와 조작 설정
  • 구매 상품 ID, 처리한 스토어 거래 ID, 저장 복구용 백업과 임시 파일
  • 밸런스 및 오류 점검용 로컬 플레이 이벤트

이 로컬 게임 데이터는 개발자 서버로 전송되지 않습니다. 운영체제 백업이 앱 데이터를 복사하는 경우 Apple 또는 Google의 설정과 정책이 적용됩니다.

6. 결제, 권한 및 삭제

유료 상품 결제는 Apple 또는 Google의 스토어 시스템이 처리하며 게임은 전체 카드 정보를 받지 않습니다. 결제 내역에는 해당 스토어 정책이 적용됩니다.

광고와 스토어 서비스에 인터넷 및 네트워크 상태 접근을 사용합니다. 카메라, 마이크, 연락처, 정밀 위치 권한은 요청하지 않습니다.

Android 앱 저장공간을 삭제하거나 iOS/Android에서 게임을 제거하면 로컬 데이터를 삭제할 수 있습니다. 게임에는 사용자 계정이나 개발자 서버가 없어 개발자가 원격으로 조회·삭제할 게임 데이터는 없습니다.

7. 아동, 보안, 보관 및 변경

게임은 아동의 개인정보를 직접 수집하도록 설계되지 않았습니다. 대상 연령에 아동이 포함되면 출시 전에 적용되는 Families/아동 규정에 맞게 광고 설정과 공개 내용을 조정합니다.

Google과 앱 스토어의 보관에는 각 사업자 정책이 적용됩니다. 로컬 데이터는 덮어쓰기, 앱 저장공간 삭제 또는 앱 제거 전까지 남습니다. SDK, 데이터 처리, 게임 기능 또는 법적 요구가 실질적으로 바뀌면 본 방침과 스토어 신고를 갱신합니다.
