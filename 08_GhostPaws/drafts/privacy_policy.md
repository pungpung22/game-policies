## 서비스와 적용 범위

게임은 진행도를 기기 안에 저장하며 게임 내 상점에서 선택형 보상 광고를
제공한다. 광고 제공과 개인정보 선택을 위해 Google Mobile Ads SDK 및 Google
User Messaging Platform(UMP)을 사용한다. 계정 로그인, 분석 SDK, 클라우드
저장, 멀티플레이, 채팅, 사용자 제작 콘텐츠는 사용하지 않는다.

## 광고 제공 과정에서 처리될 수 있는 데이터

사용자가 보상 광고를 불러오거나 시청하면 지역, 동의, 기기 설정, 노출 광고에
따라 Google 및 광고 파트너가 다음 정보를 자동으로 처리할 수 있다.

- 일반적인 위치 추정에 사용될 수 있는 IP 주소
- 광고 ID 또는 앱 범위 식별자를 포함한 기기·계정 식별자
- 앱 실행, 탭, 노출, 동영상 시청 등의 광고·제품 상호작용
- 비정상 종료, 실행 시간, 멈춤, 에너지 사용량 등의 진단·성능 정보
- 사용자에게 표시된 광고 등 광고 데이터

이 정보는 광고 제공, 광고 측정·분석, 보안, 부정행위 방지에 사용될 수 있고
암호화된 연결로 Google에 전송된다. `pungpungstudi`는 이 광고 데이터를 받는
별도 서버를 운영하지 않으며 개인정보를 직접 판매하지 않는다. Google의 처리에는
<https://policies.google.com/privacy>가 적용된다.

## 개인정보 선택과 광고 방식

앱은 필요한 지역에서 실행할 때마다 UMP로 최신 동의 상태를 확인한다. Google이
개인정보 선택 양식을 제공하면 설정 또는 상점에서 다시 열 수 있다. Android/iOS
설정에서도 광고 식별자를 제한할 수 있다. 현재 앱은 Apple 앱 추적 투명성(ATT)
권한을 요청하지 않으며 해당 권한 없이 IDFA를 의도적으로 사용하지 않는다.

광고 시청은 선택 사항이다. 보상 광고 1회 완료 시 생선쿠키 20개를 지급하며 UTC
날짜 기준 하루 최대 3회까지 받을 수 있다. 보상 완료 전에 닫으면 재화를 지급하지
않는다. 배너, 전면, 앱 시작, 강제 부활, 레벨업 리롤 광고는 사용하지 않는다.

## 로컬 게임 및 구매 데이터

진행도, 해금, 설정, 플레이 기록, 로컬 텔레메트리, 재화, 보상 광고 완료 날짜와
횟수, 구매 상품 ID, 처리한 스토어 거래 ID는 기기 안에 저장된다. 로컬 게임
데이터는 `pungpungstudi` 서버로 전송되지 않는다. 결제는 Apple 또는 Google이
처리하며 앱은 전체 카드 정보를 받지 않는다.

앱 저장공간을 삭제하거나 앱을 제거하면 로컬 데이터를 지울 수 있다. Apple 또는
Google이 보유한 결제 내역에는 해당 스토어의 정책이 적용된다.

## 권한, 아동, 보관과 변경

광고와 스토어 서비스에 인터넷 및 네트워크 상태 접근을 사용한다. 카메라, 마이크,
연락처, 정밀 위치 권한은 요청하지 않는다. 게임은 아동의 개인정보를 직접
수집하도록 설계되지 않았다. 대상 연령에 아동이 포함되면 출시 전에 적용되는
Families/아동 규정에 맞게 광고 설정과 공개 내용을 조정한다.

Google과 앱 스토어의 데이터 보관에는 각 사업자의 정책이 적용된다. 로컬 데이터는
덮어쓰기, 앱 저장공간 삭제 또는 앱 제거 전까지 남는다. SDK, 데이터 처리 또는
게임 서비스가 실질적으로 변경되면 이 방침을 갱신한다.
