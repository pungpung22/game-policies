## 핵심 고지

현재 공개 예정 빌드는 계정을 요구하지 않으며 개인정보나 기기 데이터를 개발자 서버 또는 제3자에게 수집·전송·판매·공유하지 않는다. 게임 진행 정보, 설정, 로컬 플레이 이벤트는 이용자의 기기에만 저장된다.

## 수집하지 않는 정보

- 이름, 이메일, 전화번호, 생년월일, 계정 정보
- 연락처, 사진, 카메라·마이크 데이터
- 정확한 위치 또는 대략적인 위치
- 광고·기기 식별자, IP 주소, 쿠키
- 결제 정보, 영수증 또는 거래 식별자
- 광고, 분석, 충돌 보고 또는 원격 진단 데이터

현재 광고, 분석, 충돌 보고, 계정, 클라우드 저장 또는 실제 인앱 결제 SDK가 포함되어 있지 않다.

## 기기에 저장되는 정보

- 가상 재화 잔액과 획득·사용 결과
- 고양이, 무기, 유물, 맵, 스티커 및 메타 업그레이드 상태
- 최고 기록, 런 결과, 보스 진행과 보상 사용 횟수
- 음량, 자동 공격, 접근성, UI 및 조작 설정
- 저장 무결성을 위한 저장본, 임시 파일과 백업 파일
- 밸런스와 오류 점검용 로컬 플레이 이벤트 JSONL

이 정보는 개발자에게 전송되지 않는다. 운영체제 백업이 앱 데이터를 복사하는 경우 해당 플랫폼의 설정과 정책이 적용된다.

## 권한

현재 게임 기능은 위치, 연락처, 카메라, 마이크, 사진 보관함 또는 추적 권한을 사용하거나 요청하지 않는다. Android 인터넷 권한도 비활성화되어 있다. iOS 빌드 파일에 호환용 권한 설명 키가 포함되더라도 관련 API와 권한 요청을 실행하지 않으면 데이터에 접근하지 않는다.

## 광고와 결제

코드에 향후 보상 광고와 결제를 연결하기 위한 인터페이스가 있으나, 릴리스 빌드에는 실제 제공자가 없다. 개발용 모형은 릴리스에서 보상이나 구매를 만들지 않는다. 실제 광고 또는 결제를 추가하기 전에 공개 방침, 스토어 신고, 동의와 복원 흐름을 갱신한다.

## 보유와 삭제

로컬 데이터는 이용자가 앱 데이터 또는 게임을 삭제할 때까지 남을 수 있다. Android에서는 앱 저장공간 삭제 또는 제거, iOS에서는 게임 삭제로 제거한다. 플랫폼 백업 사본은 각 플랫폼 설정에서 관리한다. 서버와 계정이 없으므로 개발자가 원격 조회·삭제할 데이터는 없다.

## 아동, 보안과 변경

게임은 연령을 확인하거나 아동의 개인정보를 의도적으로 수집하지 않는다. 로컬 저장의 손상을 줄이기 위해 백업·임시 저장 방식을 사용하지만 모든 데이터 손실을 방지할 수는 없다. SDK나 데이터 처리가 달라지면 새 빌드 제출 전에 본 방침과 스토어 신고를 갱신하고 필요한 안내·동의를 제공한다.
