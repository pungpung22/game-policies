개발자 표기는 항상 **`pungpungstudi`**를 사용한다. `Pungpung Studio`, `pungpungstudio` 또는 다른 변형을 사용하지 않는다.

공개 URL:

- 개인정보처리방침: `https://pungpung22.github.io/game-policies/08_GhostPaws/privacy/`
- 이용약관: `https://pungpung22.github.io/game-policies/08_GhostPaws/terms/`
- 지원: `https://pungpung22.github.io/game-policies/08_GhostPaws/support/`

현재 릴리스 기준:

- 패키지/Bundle ID: `com.pungpung22.ghostpaws`
- 계정 및 개발자 서버 없음
- 광고·분석·충돌 보고 SDK 없음
- 실제 인앱 결제 없음
- Android 인터넷 권한 비활성화
- 진행도와 로컬 텔레메트리는 기기 안에만 저장
- iOS App Privacy: Data Not Collected / Tracking No
- Google Play Data safety: 수집 없음 / 공유 없음

파일:

- `privacy_policy.md`: 공개 개인정보처리방침의 검토용 원문
