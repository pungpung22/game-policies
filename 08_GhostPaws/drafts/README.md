# Ghost Paws 정책 자료

이 디렉터리는 `냥벽수비대: 고스트 포우즈(Ghost Paws)`의 스토어 제출과 정책 유지보수를 위한 내부 문서다.

개발자 표기는 항상 **`pungpungstudi`**를 사용한다. `Pungpung Studio`, `pungpungstudio` 또는 다른 변형을 사용하지 않는다.

공개 URL:

- 개인정보처리방침: `https://pungpung22.github.io/game-policies/08_GhostPaws/privacy/`
- 이용약관: `https://pungpung22.github.io/game-policies/08_GhostPaws/terms/`
- 지원: `https://pungpung22.github.io/game-policies/08_GhostPaws/support/`

현재 릴리스 기준:

- 패키지/Bundle ID: `com.pungpungstudi.ghostpaws`
- 계정 및 개발자 서버 없음
- 상점 선택형 Google Mobile Ads 보상 광고와 UMP 포함
- Android Firebase Remote Config와 Crashlytics/Crashlytics NDK 포함
- Firebase Google Analytics 사용 안 함
- iOS Firebase 앱/config 등록 완료, 네이티브 SDK 브리지는 출시 전 블로커
- 실제 인앱 결제 없음
- Android 인터넷/네트워크 상태 접근 활성화
- 진행도와 로컬 텔레메트리는 기기 안에만 저장
- iOS App Privacy: Google Mobile Ads 및 향후 Firebase iOS 실제 빌드 기준 / 현재 Tracking No 계획
- Google Play Data safety: Google Mobile Ads + Remote Config + Crashlytics 기준으로 제출
- Google Play Ads declaration: Yes

파일:

- `privacy_policy.md`: 공개 개인정보처리방침의 검토용 원문
- `terms_of_service.md`: 공개 이용약관의 검토용 원문
- `sdk_data_inventory.md`: 코드 및 SDK 데이터 처리 목록
- `store_privacy_disclosure_guide.md`: Apple/Google 스토어 입력 가이드
- `consent_flow.md`: 현재 동의 판단과 향후 광고 SDK 추가 체크리스트

이 문서는 법률 자문이 아니다. 출시 국가, 대상 연령, 실제 포함 SDK와 빌드 동작이 바뀌면 제출 전에 개발자와 필요한 전문 검토자가 다시 확인해야 한다.
