게임 오류, 광고 보상, 구매 복원, 정책 또는 스토어 관련 문의는 이메일이나 공개 지원 이슈를 이용해 주세요.

문의에 포함할 내용

  • 앱 버전과 빌드 번호
  • 기기 모델 및 Android 또는 iOS 버전
  • 문제가 발생한 스테이지와 화면
  • 문제를 다시 발생시킬 수 있는 순서
  • 개인정보가 포함되지 않은 스크린샷

저장 데이터와 재화

진행 정보, 물고기, 아이템과 컬렉션 상태는 현재 기기에 저장됩니다. 앱을 지우거나 저장공간을 초기화하기 전에 문제 상황을 먼저 기록해 주세요. 개발자는 원격 저장 데이터에 접근할 수 없어 삭제된 진행이나 소비성 재화를 복원하지 못할 수 있습니다.

광고 보상 문제

광고를 끝까지 시청했는데 보상이 지급되지 않았다면 광고 종류, 발생 시각, 당시 네트워크 상태와 화면을 알려 주세요. 같은 광고를 반복해서 실행하거나 앱을 즉시 삭제하기 전에 문의하면 원인 확인에 도움이 됩니다.

구매와 복원

영구 펭귄 컬렉션은 설정의 구매 복원에서 동일한 스토어 계정으로 복원할 수 있습니다. 소비성 물고기는 일반적으로 복원되지 않습니다. 결제만 되고 상품이 지급되지 않았다면 재구매하지 말고 주문번호 일부와 상품명을 이메일로 보내 주세요. 카드 번호나 영수증 전체 이미지는 보내지 마세요.

개인정보 및 삭제

개발자 서버나 사용자 계정이 없으므로 개발자가 원격으로 보유하는 게임 진행 데이터는 없습니다. Android 앱 저장공간을 삭제하거나 iOS/Android에서 게임을 제거하면 로컬 데이터를 삭제할 수 있습니다. 광고 또는 Firebase 관련 문의는 개인정보처리방침을 확인해 주세요.

공개 이슈 주의사항

GitHub 이슈는 공개됩니다. 실명, 이메일, 전화번호, 주소, 비밀번호, 결제 카드, 영수증 전체, Apple ID 또는 Google 계정 정보를 게시하지 마세요. 민감한 내용은 이메일을 이용해 주세요.
